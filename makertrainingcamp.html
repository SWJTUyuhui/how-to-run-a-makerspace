How to run a makerspace

some guides about instrument,activity and operation

  • introduction

    • 创客训练营是一个集交流学习、研讨制作为一体的创客教育活动。在这里，来自社会、学校的创客们汇聚在一起，一起学习交流，并通过一个个有趣的工作坊，动手制作，学会一些时下必会技能。

  • 活动筹备

    • 距离活动还有一个月---->确定活动主题，费用预算

    -------------------------->寻找企业，政府赞助

    距离活动还有两周---->项目经费到达各负责人（活动材料采购经费，宣传用品经费，茶点经费，礼品经费）

    -------------------------->设计活动赛海报，印刷，张贴

    -------------------------->微信公众号，官网，交大创客qq群、微信群，科技局微信，中国移动，推送比赛信息

    -------------------------->嘉宾评委邀请，主持人预约，讲师邀请（训练营项目确定）

    -------------------------->场地确定，经费到位

    -------------------------->购买活动材料；购买嘉宾、评委、讲师、志愿者礼品

    距离活动还有一周---->志愿者招募、安排，食宿预约

    距离活动还有1天---->场地布置，网络布置，购买茶点

    距离活动还有0天---->拍照，现场组织

    -------------------------->整理照片，存百度云盘，撰写、发布活动报道

    活动结束---->打扫场地，整理设备
